## What is the Python Debugger?  

- interactive source code debugger for Python programs.


### How do we use it?  

```
$ python3 -m pdb myscript.py  
```

OR  

```python
>>> import pdb  
>>> import myscript  
>>> pdb.set_trace()  
```


### Commands  

1. ? or help will show list of commands  
2. q = quit  
3. cl = clear  
4. b = break  
5. d = debug  
6. l = list  
7. s = step  
8. n = next
